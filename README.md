# Personal Portfolio Website

A modern, interactive personal portfolio website showcasing my projects, skills, and games collection.

## Features

- **Responsive Design** - Works seamlessly on desktop and mobile devices
- **Interactive Games** - 9 built-in mini-games (Tetris, Snake, Pong, Memory, Breakout, Flappy Bird, Tic Tac Toe, Asteroids, Simon Says)
- **Project Showcase** - Highlights my development work with live demos
- **Smooth Animations** - Modern UI with hover effects and transitions
